# PicDownload.

抖音图库解析下载

新添加GUI界面
GUI版本支持 视频解析下载
Releases下载最新版本 运行exe文件即可

运行TuKuDownload.py 文件

记得导入库
pip install pillow
pip install tqdm

在 conf.ini 可以直接粘贴多个url（如果运行失败，请手动把换行符删掉）

有问题 请留言
